# 面试行为评定 - 语言表达能力

> 本模块用于 AI 评估候选人的语言表达和沟通质量，侧重帮助应届生提升面试自信度

---

## 评估维度概述

语言表达能力是面试中最直观的评估指标，直接影响面试官对候选人的第一印象。对于应届生来说，即使技术能力不错，表达不清晰也会大大降低面试通过率。

---

## 细分指标

### 1. 语言流畅度

**评估标准：**

| 等级 | 分数 | 行为锚点 |
|------|------|---------|
| 优秀 | 9-10 | 表达自然流畅，语速适中（180-220字/分钟），几乎无卡顿和重复 |
| 良好 | 7-8 | 整体流畅，偶尔有短暂停顿但能迅速接上，不影响理解 |
| 中等 | 5-6 | 时有卡顿，有明显的"嗯""啊""那个"等填充词，但能完成表达 |
| 较差 | 3-4 | 频繁卡顿、重复同一句话、长时间停顿（>5秒），影响信息传达 |
| 很差 | 1-2 | 几乎无法完成完整句子，严重卡顿或完全沉默 |

**关键检测指标：**
- 填充词频率（"嗯"、"啊"、"那个"、"就是"每分钟出现次数）
- 单次停顿时长（正常 < 2秒，提示关注 > 3秒，严重 > 5秒）
- 语句完整率（能否完成一个完整的语义表达）
- 重复表述次数（同一意思反复说多次）

**给应届生的改进建议：**
- 允许短暂停顿整理思路，不必急于填充
- 练习用"首先…其次…最后…"等结构词引导表达
- 提前准备高频问题的回答框架，降低临场紧张

---

### 2. 回答结构性

**评估标准：**

| 等级 | 分数 | 行为锚点 |
|------|------|---------|
| 优秀 | 9-10 | 使用明确的回答框架（总-分-总 / STAR），层次分明，逻辑递进 |
| 良好 | 7-8 | 有基本结构，能区分主次，但偶尔结构不够紧凑 |
| 中等 | 5-6 | 有尝试组织结构，但层次不够清晰，信息交叉重叠 |
| 较差 | 3-4 | 想到什么说什么，缺乏组织，核心信息淹没在细节中 |
| 很差 | 1-2 | 完全无结构，语义断裂，前后不连贯 |

**结构化回答模板：**
- 技术问题：「结论先行 → 原理展开 → 举例说明 → 对比延伸」
- 项目经验：「STAR 法则（背景 → 任务 → 行动 → 结果）」
- 开放问题：「观点 → 论据1 → 论据2 → 总结」

**检测规则：**
- 是否有明确的开头引导语（"这个问题我从三个方面来回答"）
- 是否使用结构标记词（"第一"、"第二"、"总结来说"）
- 结论是否在前 30 秒内给出

---

### 3. 专业术语使用

**评估标准：**

| 等级 | 分数 | 行为锚点 |
|------|------|---------|
| 优秀 | 9-10 | 术语使用准确且自然，能在专业表达和通俗解释间自由切换 |
| 良好 | 7-8 | 术语基本准确，偶尔有不够精确但不影响理解 |
| 中等 | 5-6 | 使用部分术语但有时混淆概念或用错语境 |
| 较差 | 3-4 | 很少使用专业术语，或大量术语使用错误 |
| 很差 | 1-2 | 无法使用任何专业术语，全部口语化表达 |

**检测要点：**
- 术语与问题领域的匹配度
- 是否能用自己的话解释术语（理解 vs 背诵）
- 中英文术语的使用一致性

---

### 4. 回答简洁度

**评估标准：**

| 等级 | 分数 | 行为锚点 |
|------|------|---------|
| 优秀 | 9-10 | 回答精炼有力，每句话都有信息量，无冗余内容 |
| 良好 | 7-8 | 基本简洁，偶有少量冗余但整体紧凑 |
| 中等 | 5-6 | 有些啰嗦，存在重复表达或无关内容，但核心信息在 |
| 较差 | 3-4 | 大量冗余内容，跑题严重，核心要点不突出 |
| 很差 | 1-2 | 严重跑题或过度展开无关细节，无法找到有价值的信息 |

**时间参考：**
- 简单概念问题：理想回答 1-2 分钟
- 原理性问题：理想回答 2-3 分钟
- 项目经验问题：理想回答 3-5 分钟
- 超过参考时间 50% 且信息密度低，扣分

---

## AI 评估提示词模板

```
评估候选人在以下维度的语言表达能力（每项 1-10 分）：
1. 语言流畅度：是否流畅自然，有无频繁卡顿和填充词
2. 回答结构性：是否有清晰的回答框架和逻辑层次  
3. 专业术语使用：术语是否准确，是否真正理解而非死记硬背
4. 回答简洁度：信息密度是否高，是否跑题或冗余

请为每一项给出具体分数和改进建议，语气鼓励而具体。
```
