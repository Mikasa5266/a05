# 面试行为评定 - 非语言行为（视频面试适用）

> 本模块评估候选人在视频/音频面试中的非语言行为信号

---

## 评估维度概述

非语言行为（包括声音特征和视觉表现）在面试中传递的信息量往往超过语言内容本身。研究表明面试官对候选人的印象中约55%来自非语言信号。对于应届生来说，提升非语言表现是快速增强面试效果的捷径。

---

## 细分指标

### 1. 声音特征

**评估标准：**

| 等级 | 分数 | 行为锚点 |
|------|------|---------|
| 优秀 | 9-10 | 音量适中清晰，语调有变化（重点处加重/升调），语速适当（180-220字/分钟） |
| 良好 | 7-8 | 整体清晰可听，语调有少量变化，语速偶有波动但在合理范围 |
| 中等 | 5-6 | 声音偏小或偏快，语调较平但可以听清，信息可理解 |
| 较差 | 3-4 | 声音很小/模糊/很快，语调完全平直无起伏（像背诵），影响理解 |
| 很差 | 1-2 | 声音几乎无法听清，或语速过快/过慢导致无法正常交流 |

**细分指标：**
- **音量稳定性**：是否保持适中音量，不忽大忽小
- **语调变化**：重点内容是否有语气加强（避免全程平调像背答案）
- **语速控制**：是否根据内容复杂度调整语速（解释原理时适当放慢）
- **情绪表达**：声音是否传递出热情和投入（vs 冷淡疲惫）

---

### 2. 表情与眼神（视频面试）

**评估标准：**

| 等级 | 分数 | 行为锚点 |
|------|------|---------|
| 优秀 | 9-10 | 表情自然放松，有适度微笑，眼神注视摄像头表现出专注和尊重 |
| 良好 | 7-8 | 表情基本自然，大部分时间能看摄像头，偶有低头思考 |
| 中等 | 5-6 | 表情略显紧张，眼神游移但能回到摄像头方向 |
| 较差 | 3-4 | 面无表情或持续紧绷，很少看摄像头，疑似在看提词 |
| 很差 | 1-2 | 表情明显焦虑痛苦，完全不看摄像头，严重影响面试氛围 |

**注意事项（对应届生友好）：**
- 看摄像头 ≠ 一直盯着：适当移开思考是正常的
- 紧张的表情不是减分项，关键是是否在尝试调整
- 视频面试可以将笔记贴在摄像头旁边辅助

---

### 3. 回答时长控制

**评估标准：**

| 问题类型 | 理想时长 | 过短 | 过长 |
|---------|---------|------|------|
| 概念定义 | 30-90秒 | < 15秒 | > 2分钟 |
| 原理解释 | 1-3分钟 | < 30秒 | > 4分钟 |
| 项目经验 | 2-5分钟 | < 1分钟 | > 7分钟 |
| 自我介绍 | 1-3分钟 | < 30秒 | > 5分钟 |

| 等级 | 分数 | 行为锚点 |
|------|------|---------|
| 优秀 | 9-10 | 能根据问题类型自然调控回答时长，详略得当 |
| 良好 | 7-8 | 大部分回答时长合理，偶有过长或过短 |
| 中等 | 5-6 | 时长控制意识不强，部分回答明显过长或过短 |
| 较差 | 3-4 | 经常性地简答草草了事，或冗长无重点 |
| 很差 | 1-2 | 一两句话打发，或长篇大论完全不着重点 |

---

### 4. 肢体语言（视频面试）

**评估标准：**

| 等级 | 分数 | 行为锚点 |
|------|------|---------|
| 优秀 | 9-10 | 坐姿端正自然，适度手势辅助表达，背景整洁 |
| 良好 | 7-8 | 坐姿基本端正，少量手势，环境较合适 |
| 中等 | 5-6 | 坐姿偶有松懈，很少有手势，环境一般 |
| 较差 | 3-4 | 明显驼背/斜靠，环境嘈杂或杂乱 |
| 很差 | 1-2 | 躺着/随意姿势，环境完全不适合面试 |

**视频面试环境建议：**
- 背景简洁干净（纯色墙壁最佳）
- 光线充足且正面打光（避免逆光）
- 摄像头在眼睛水平线上方（略俯视最自然）
- 网络稳定，提前测试

## AI 音频分析指标（技术实现参考）

```
从音频流中提取以下特征用于非语言行为评估：
1. 语速（字/分钟）：正常范围 160-240
2. 停顿次数和时长：统计 > 2秒的停顿
3. 填充词频率：统计"嗯""啊""那个"等
4. 音量均值和方差：检测音量稳定性
5. 音高（基频 F0）频率和变化：检测语调单调性
6. 信噪比：检测环境噪音
7. 静音比例：总沉默时间/总时间
```

## AI 评估提示词模板

```
评估候选人在以下非语言维度的表现（每项 1-10 分）：
1. 声音特征：音量、语调、语速是否适当
2. 表情眼神：是否自然专注（若为视频面试）
3. 时长控制：回答长度是否与问题复杂度匹配
4. 整体仪态：环境、坐姿和精神状态

注意：本系统面向应届生，对于紧张导致的非语言问题应理解包容，
评估重点是帮助其意识到问题并给出改进方法，而非扣分惩罚。
```
