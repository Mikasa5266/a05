## Q1: 监督学习、无监督学习和强化学习的区别

**规范答案：**

| 类别 | 数据 | 目标 | 典型算法 |
|------|------|------|---------|
| **监督学习** | 有标签数据 (X, Y) | 学习 X→Y 的映射 | 线性回归、SVM、随机森林、神经网络 |
| **无监督学习** | 无标签数据 (X) | 发现数据中的结构 | K-Means、DBSCAN、PCA、自编码器 |
| **半监督学习** | 少量标签 + 大量无标签 | 利用无标签数据提升效果 | 伪标签、一致性正则 |
| **强化学习** | 环境反馈（奖励） | 学习最优策略 | Q-Learning、DQN、PPO、A3C |

**监督学习细分：**
- 回归（输出连续值）：房价预测、股票价格
- 分类（输出离散值）：垃圾邮件、图像识别

**得分点：**
- [ ] 正确区分三种学习范式的核心差异
- [ ] 知道每种范式的典型算法
- [ ] 给出合理的应用场景
- [ ] 理解有标签 vs 无标签的本质影响

**加分项：**
- 提到自监督学习（BERT、MAE 的预训练方式）
- 分析数据标注成本与半监督学习的价值
- 提到对比学习（Contrastive Learning）

**追问方向：**
- 什么是过拟合和欠拟合？如何解决？
- 偏差-方差权衡是什么？
- 交叉验证有哪些方式？

## Q2: 常见分类算法对比

**规范答案：**

| 算法 | 原理 | 优点 | 缺点 | 适用场景 |
|------|------|------|------|---------|
| **逻辑回归** | Sigmoid + 交叉熵 | 简单高效、可解释 | 线性边界 | 二分类、CTR 预估 |
| **SVM** | 最大化间隔超平面 | 高维有效、泛化好 | 大规模数据慢 | 小样本、高维 |
| **决策树** | 递归分裂（信息增益/基尼） | 可解释、非参数 | 易过拟合 | 特征重要性分析 |
| **随机森林** | 多棵决策树 Bagging | 稳定、抗过拟合 | 不易解释 | 通用分类/回归 |
| **XGBoost** | 梯度提升 + 正则化 | 精度高、有正则 | 调参复杂 | 结构化数据竞赛 |
| **神经网络** | 多层非线性变换 | 拟合能力强 | 需要大数据 | 图像、NLP、语音 |

**模型选择原则：**
1. 数据量小 + 特征工程好 → 传统 ML（XGBoost、SVM）
2. 数据量大 + 原始特征 → 深度学习
3. 需要可解释性 → 逻辑回归、决策树

**得分点：**
- [ ] 至少了解 4 种分类算法的原理
- [ ] 知道各算法的适用场景和局限性
- [ ] 能区分 Bagging 和 Boosting
- [ ] 有模型选择的思路

**加分项：**
- 深入分析 XGBoost 的目标函数和正则项
- 对比 XGBoost 和 LightGBM 的区别
- 提到集成学习的 Stacking 方法

**追问方向：**
- SVM 的核函数有哪些？什么时候用非线性核？
- 随机森林中的随机体现在哪里？
- XGBoost 的损失函数是怎么推导的？

## Q3: 模型评估指标

**规范答案：**

**分类指标：**
| 指标 | 公式 | 侧重点 |
|------|------|--------|
| Accuracy | (TP+TN) / 总数 | 整体正确率，不适合不平衡数据 |
| Precision | TP / (TP+FP) | 预测为正的准确率（关注误报） |
| Recall | TP / (TP+FN) | 实际为正的召回率（关注漏报） |
| F1 Score | 2·P·R / (P+R) | Precision 和 Recall 的调和平均 |
| AUC-ROC | ROC 曲线下面积 | 综合评估，不受阈值影响 |

**不同场景的指标选择：**
- 诈骗检测（漏报代价高）→ 侧重 Recall
- 垃圾邮件（误报代价高）→ 侧重 Precision
- 综合评估 → F1 或 AUC

**回归指标：**
- MSE（均方误差）：对大误差敏感
- MAE（平均绝对误差）：对异常值鲁棒
- R²（决定系数）：解释方差比例

**得分点：**
- [ ] 理解混淆矩阵和 TP/FP/TN/FN
- [ ] 正确计算 Precision、Recall、F1
- [ ] 知道不平衡数据下 Accuracy 的局限
- [ ] 能根据业务场景选择合适指标

**加分项：**
- 能画出 ROC 曲线并解释 AUC 的含义
- 提到 PR 曲线（适用于不平衡数据）
- 提到 kappa 系数和混淆矩阵的热力图

**追问方向：**
- 如何处理样本不平衡问题？
- AUC 为什么不受阈值和样本不平衡影响？
- 多分类场景下如何计算 F1？

## Q4: 特征工程的方法

**规范答案：**

**特征处理：**
1. **数值特征**：归一化（MinMax 到 [0,1]）、标准化（Z-score）、对数变换
2. **类别特征**：One-Hot 编码、Label 编码、Target 编码、Embedding
3. **缺失值处理**：均值/中位数/众数填充、KNN 填充、标记为新类别
4. **异常值处理**：IQR/3σ 方法检测、截断或变换

**特征选择：**
1. **过滤法**：方差筛选、相关系数、卡方检验、互信息
2. **包裹法**：递归特征消除（RFE）、前向/后向选择
3. **嵌入法**：L1 正则化（Lasso）、树模型特征重要性

**特征构造：**
- 交叉特征：`age × income`
- 统计特征：均值、方差、分位数
- 时序特征：滑动窗口统计、趋势特征

**得分点：**
- [ ] 知道数值和类别特征的常见处理方式
- [ ] 至少了解两类特征选择方法
- [ ] 理解归一化和标准化的区别和适用场景
- [ ] 知道为什么要做特征工程

**加分项：**
- 提到 Embedding 替代 One-Hot 的优势（高基数特征）
- 讨论特征交叉在推荐系统中的应用
- 提到自动特征工程工具（Featuretools、AutoFeat）

**追问方向：**
- 什么时候用归一化？什么时候用标准化？
- PCA 降维的原理是什么？
- 如何处理高基数类别特征？

## Q5: 正则化与过拟合

**规范答案：**

**过拟合的表现：**
训练集上表现好，验证/测试集上表现差，模型学到了数据中的噪声。

**解决过拟合的方法：**

| 方法 | 原理 | 适用场景 |
|------|------|---------|
| **L1 正则化** | 添加权重绝对值惩罚项 | 特征选择（稀疏解） |
| **L2 正则化** | 添加权重平方惩罚项 | 防止权重过大 |
| **Dropout** | 训练时随机丢弃神经元 | 深度学习 |
| **数据增强** | 扩充训练数据 | 图像、NLP |
| **早停（Early Stopping）** | 验证集损失不再下降时停止 | 通用 |
| **交叉验证** | 多折验证评估泛化能力 | 通用 |
| **降低模型复杂度** | 减少层数/参数量 | 模型过于复杂时 |

**欠拟合的解决：**
- 增加模型复杂度
- 增加特征
- 减少正则化强度
- 训练更长时间

**L1 vs L2 正则化：**
- L1 产生稀疏解（部分权重为 0），可做特征选择
- L2 让权重趋向于小值但不为零，更平滑

**得分点：**
- [ ] 理解过拟合和欠拟合的概念
- [ ] 至少知道 4 种防止过拟合的方法
- [ ] 理解 L1 和 L2 正则化的区别
- [ ] 知道 Dropout 的原理

**加分项：**
- 从贝叶斯角度解释正则化（L2 = 高斯先验）
- 分析偏差-方差分解
- 提到 Batch Normalization 的正则化效果

**追问方向：**
- Dropout 在测试时需要做什么？为什么？
- Batch Normalization 是怎么工作的？
- 如何选择正则化系数？

## Q6: 数学基础（概率统计与线性代数）

**规范答案：**

**概率统计：**
- **贝叶斯定理**：P(A|B) = P(B|A)·P(A) / P(B)，朴素贝叶斯分类器的基础
- **最大似然估计（MLE）**：找使观测数据出现概率最大的参数
- **最大后验估计（MAP）**：MLE + 先验分布，等价于加正则化
- **常见分布**：正态分布、伯努利分布、多项分布、泊松分布

**线性代数：**
- **特征值和特征分解**：PCA 降维的数学基础
- **SVD（奇异值分解）**：矩阵分解，应用于推荐系统和降维
- **矩阵运算**：矩阵乘法、转置、逆矩阵、行列式
- **梯度**：标量对向量的偏导数，梯度下降的基础

**信息论：**
- **信息熵**：H(X) = -Σ p(x) log p(x)，衡量不确定性
- **交叉熵**：H(p,q) = -Σ p(x) log q(x)，分类损失函数
- **KL 散度**：衡量两个分布的差异，D(p||q) = Σ p(x) log(p(x)/q(x))

**得分点：**
- [ ] 理解贝叶斯定理及其应用
- [ ] 知道 MLE 的基本思想
- [ ] 了解特征值分解/SVD 的概念
- [ ] 理解交叉熵损失函数的来源

**加分项：**
- 推导逻辑回归的损失函数（从 MLE 推导）
- 解释 SVD 在推荐系统中的应用
- 分析 KL 散度的非对称性

**追问方向：**
- 为什么分类问题用交叉熵而不用 MSE？
- PCA 的数学推导过程？
- 梯度消失和梯度爆炸是什么？如何解决？
