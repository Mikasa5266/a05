# 腾讯云前端一面面经（含高频题 + 复盘）

> 来源: 校友社区 (作者: 123)
> 发布时间: 2026-03-06 21:19:21

## 面试信息
- **公司**: 腾讯
- **岗位**: 前端工程师（云产品方向）

## 面试流程
一面：基础 + 项目深挖 + 场景题；
后续：二面（技术面）、HR 面（流程中）

## 高频面试题
Vue3 的响应式原理（Proxy 与 Object.defineProperty 的区别）；
如何封装一个可复用的网络请求库（拦截器、重试、缓存）；
前端工程化：Vite 与 Webpack 的核心差异；
场景题：如何优化云控制台中大量表格的渲染性能；
跨域问题的解决方案及原理；
对前端适配云产品控制台的理解。

## 复盘建议
不足：对云控制台的性能场景思考不够深入，工程化配置细节回答不够流畅；
改进：重点复习 Vue3 源码片段，多做大型后台系统的性能优化案例，熟悉 Vite 插件开发；
推荐资料：《Vue.js 设计与实现》、Vite 官方文档、腾讯云控制台开源组件库。
