# 字节跳动前端三面面经（含高频题 + 复盘）

> 来源: 校友社区 (作者: 123)
> 发布时间: 2026-03-06 21:07:04

## 面试信息
- **公司**: 字节跳动
- **岗位**: 前端工程师

## 面试流程
一面：基础 + 手撕代码（JS/React）+ 算法题；
二面：项目深挖 + 系统设计（前端工程化 / 性能优化）+ 场景题；
三面：HR 面 + 职业规划 + 综合评估

## 高频面试题
React diff 算法原理及优化点；
浏览器渲染流程与重排重绘；
前端缓存策略（强缓存 / 协商缓存）；
实现一个防抖 / 节流函数；
算法：二叉树层序遍历、最长回文子串；
如何优化大型 React 应用的性能；
对前端工程化的理解（构建、CI/CD）。

## 复盘建议
不足：算法题思路不够快，对前端工程化细节掌握不深；
改进：多刷 LeetCode，重点复习 Webpack/Vite 等构建工具原理，多看大厂开源项目；
推荐资料：《React 设计原理》、《浏览器工作原理与实践》、字节前端 youtu.be 系列文章。
